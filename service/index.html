Penjelasan singkat

Beberapa sistem service, sebagai pendukung terhadap service utama. Dibangun dengan ExpressJS, Deno, dan Golang

Penjabaran
Api : - Sebagai salah satu service backend utama yang menangani Rekening, Kepegawaian, Pelanggan, dan Skematik
App : - Sistem lama (monolith) yang sudah tidak terpakai, tapi tetap dibutuhkan, sehingga tidak saya matikan
Auth : - Service backend yang menangani fungsi enkripsi, otentikasi dan hak akses user
Botpdam dan Botumum : - Service Bot Telegram untuk internal dan pelanggan
Cron : - Service yang menangani beberapa fungsi untuk keperluan otomasi
DSML : - Service untuk menangani pembaca / pencatat meter
Notif : - Service yang berfokus menangani dan mengolah data notifikasi baik itu lewat SMS, WA, Telegram, atau Notifikasi lewat Aplikasi Android "Banyu"
Pay : - Service payment gateway untuk pembayaran yang dilakukan oleh PPOB atau mitra pembayaran, dengan total mitra 11 biller
Virtual Account : - Service yang fokus menangani proses pembuatan dan pembayaran lewat Virtual Account (saat ini hanya BRIVA)
Assets : - Service backend yang berjalan di mesin server kedua, untuk kepeluan serving asset baik itu gambar, video, atau file
Banyu : - Service backend yang berjalan di mesin server kedua, untuk aplikasi Android "Banyu"
Banyu : - Service backend yang berjalan di mesin server kedua, yang berfokus pada upload file atau data dari mobile maupun web
