Penjelasan singkat

Aplikasi utama, yang saya buat dan kembangan sendiri (fullstack) di PDAM Kabupaten Batang guna menggantikan aplikasi desktop yang sudah tidak dikembangkan oleh pihak yang bersangkutan. Awal mulanya pada tahun 2016 saya menggembangkan aplikasi ini menggunakan monolithic design, dimana seluruh aplikasi menjadi satu dengan Framework express.js dengan template jade (pug) dengan library jQuery dan Vue.js. Namun pada tahun 2019, bertepatan saya membuat aplikasi mobile Android "Banyu" untuk perusahaan, munculah ide untuk memisah-misahkan seluruh service menjadi lebih kecil agar perawatan dan pengembangan lebih mudah kedepannya, atau istilahnya kerennya dengan sebutan "Microservice". Juga untuk teknologi front end saya mempercayakan menggunakan react dengan framework NextJS dengan SSRnya. Perubahan arsitektur dari monolith ke microservice rampung seluruhnya pada bulan Desember 2021, baik dari front end maupun backend. Aplikasi ini mencakup hampir sebagian besar keperluan pengolahan data baik dari rekening pelanggan sampai dengan manajemen kepegawaian.

Teknologi yang dipakai
Front End : - React
- Next.JS
- Tailwind CSS
Template : - Full Custom
Back End : - NodeJS
- ExpressJS
- Golang
Database : - MySQL (Utama)
- PostgreSQL (GIS)
- Redis (Caching)
- MongoDB (User)
